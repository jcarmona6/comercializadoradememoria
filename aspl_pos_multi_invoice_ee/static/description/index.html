Odoo 15 POS Multi Invoice (Enterprise)

This module provide the information about POS Multi Invoice .

Note: This module is tested and working in Odoo Enterprise version . And we are supporting Ubuntu OS and not Windows OS.

Available for Community

How to use

  • We can configure POS Multi Invoice settings here.


  • In below screen, we can see that there is no invoice created for these POS Orders.


  • Menu for POS Invoice is at "Point Of Sale > Orders > POS Invoice".
  • We can easily filter POS Orders by Dates and Customer , which are "Paid" state.


  • Invoice will be created in "Draft" state.
  • In settings, If Multi Invoices is selected, then the combined invoice will be created for the selected POS Orders.
  • In settings, If Single Invoice is selected, then the separate invoice will be created for the selected POS Orders.


  • After creating Invoice, Invoice and Status fields value of selected POS Orders will be changed from "Paid" to "Invoiced" state.


  • If Invoice get Cancelled, It will update both fields into POS Orders .Invoice should be blank and status will be "Paid" state.
